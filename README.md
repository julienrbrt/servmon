# Servmon

KISS server monitoring tool with email alerts.
For those who want to keep it simple instead of using Prometheus, Grafana, and Alertmanager.

Monitors:

- [x] CPU
- [x] Memory
- [x] HTTP Health check
- [x] Disk Usage
- [ ] Disk Write/Read
- [ ] Docker

## Installation

```bash
go install github.com/julienrbrt/servmon@latest
```

## How to use

```bash
servmon --help
```
